Catatan Keuangan
Harian

Hubungkan ke Google Sheets agar data tersimpan aman dan bisa diakses dari HP maupun PC kapan saja.

1

Buka sheets.new → buat spreadsheet baru, beri nama bebas (misal: Keuangan Harian).

2

Klik Extensions → Apps Script. Paste seluruh isi file Code.gs yang disertakan, lalu simpan.

3

Klik Deploy → New Deployment. Pilih type Web App, set Execute as: Me dan Who has access: Anyone. Klik Deploy & copy URL-nya.

4

Paste URL deployment di bawah ini, lalu klik Hubungkan.

URL tersimpan di browser kamu. Data tidak pernah meninggalkan Google akunmu.
